# ⎈ CKA Revision Guide

A free, interactive revision guide for the **Certified Kubernetes Administrator (CKA)** exam by the CNCF.

**Live site** → [cka-guide.pages.dev](https://cka-guide.pages.dev/)

---

## What's Inside

| Category | Topics |
|---|---|
| **Foundations** | Deploy Order · Namespace · ConfigMap · Secret · PV & PVC · Pod |
| **Workloads** | ReplicaSet · Deployment · StatefulSet · DaemonSet · Job · CronJob |
| **Networking** | Service Types (ClusterIP, NodePort, LoadBalancer, ExternalName) |
| **Scaling & Scheduling** | HPA & KEDA · VPA · Taints & Tolerations · Node Affinity |

Every topic includes:
- Concise theory with exam-relevant details
- Ready-to-use `kubectl` commands
- Complete YAML manifests you can copy-paste
- CKA exam tips and gotchas
